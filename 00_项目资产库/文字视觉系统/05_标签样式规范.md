# 标签样式规范

## 小标签

| 项 | 规范 |
|----|------|
| 字体 | 中粗圆体 / 中黑体 |
| 字号 | 正文的 70%～80% |
| 形状 | 圆角胶囊或圆角矩形 |
| 内边距 | 左右 ≥ 字高 0.8 倍 |

## 标签类型与配色

| 类型 | 底色 | 文字色 |
|------|------|--------|
| 作用 / 来源 | `#DDEBF7` | `#4A647A` |
| 正确做法 | `#DCEAD7` | `#6F8F68` |
| 误区提醒 | `#FFE0C2` | `#D85C3A` |
| 注意事项 | `#FFF1C7` | `#5A3B2E` |

## 对话气泡

- 底色 `#FFF8EF` 或 `#FFF4DD`，描边 `#9A6A4A` 1px
- 文字 `#5A3B2E`，单行不超过 14 字
- 尖角指向说话角色，不挡表情

## 禁止

- 过多标签堆叠（单页 ≤ 3 个）
- 荧光色、立体渐变标签
