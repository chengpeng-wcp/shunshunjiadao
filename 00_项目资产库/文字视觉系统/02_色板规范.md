# 色板规范

## 统一色板

| 用途 | 色值 | 说明 |
|------|------|------|
| 背景米白 | `#FFF8EF` | 画面主背景 |
| 主标题橙 | `#E98B45` | 主标题、强调 |
| 深棕文字 | `#5A3B2E` | 标题、字幕主色 |
| 奶茶棕 | `#9A6A4A` | 副标题、次要强调 |
| 正文深灰 | `#444444` | 正文、说明 |
| 辅助灰 | `#777777` | 注释、次要信息 |

## 卡片底色

| 名称 | 色值 | 适用 |
|------|------|------|
| 浅绿卡片 | `#DCEAD7` | 正确做法、科普要点 |
| 浅蓝卡片 | `#DDEBF7` | 知识解释、中性信息 |
| 浅黄卡片 | `#FFF1C7` | 提示、清单 |
| 浅橙卡片 | `#FFE0C2` | 误区、警告区 |

## 功能色

| 名称 | 色值 | 适用 |
|------|------|------|
| 提醒红橙 | `#D85C3A` | 误区、禁忌、强调词 |
| 深绿色 | `#6F8F68` | 正确提示文字 |
| 深蓝灰 | `#4A647A` | 次要信息条 |
| 铁锈红 | `#B85C38` | 严重提醒（少用） |

## 字幕专用

| 用途 | 色值 |
|------|------|
| 字幕底条 | `#FFF4DD` |
| 字幕文字 | `#5A3B2E` |
| 强调词 | `#D85C3A` |

**原则：** 低饱和、奶油暖色，禁止高饱和早教色、彩虹渐变字。
