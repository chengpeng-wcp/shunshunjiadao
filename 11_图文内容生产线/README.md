# 11_图文内容生产线

《顺顺一家育儿小剧场》图文专用流水线。模式说明见 `00_图文模式说明/`。

## 每篇核心生产链条

```
图文选题（01）
    ↓
图文脚本（02）
    ↓
小剧场页面设计（03）
    ↓
图文图片提示词（04）
    ↓
图文成图（05）
    ↓
图文发布稿（06）
    ↓
图文复盘（07）
```

| 环节 | 目录 |
|------|------|
| 模式/原则 | `00_图文模式说明` |
| 选题 | `01_图文选题库` |
| 脚本 | `02_图文脚本/{篇号}/` |
| 页面文案 | `03_小剧场页面设计/{篇号}/` P1～P6 |
| 出图提示词 | `04_图文图片提示词/{篇号}/` |
| 成图 | `05_图文成图/{篇号}/` |
| 发布 | `06_图文发布稿/{篇号}/` |
| 复盘 | `07_图文复盘/{篇号}/` |
| 模板 | `08_图文模板库` |

## 新篇命名

复制 `001_宝宝补钙误区`，改为 `002_xxx`，在 `02`～`07` 同步维护。

角色与场景资产引用 `00_项目资产库/`；画面内文字样式统一引用 `00_项目资产库/文字视觉系统/`（本目录不重复维护）。
