# 正文样式规范

## 基础样式

| 项 | 规范 |
|----|------|
| 字体 | 思源黑体 / 普惠体 Regular～Medium |
| 颜色 | `#444444`，次要 `#777777` |
| 行距 | 1.4～1.6 倍 |
| 对齐 | 左对齐为主，清单可居中 |

## 使用场景

- 图文科普正文
- 信息卡片说明段
- 视频画面内少量说明字（非字幕）

## 排版原则

- 每段不超过 3 行，单页正文不超过 80 字
- 重点词用 `#D85C3A` 或加粗，不全段标红
- 列表用圆点或数字，层级不超过 2 级
- 留白充足，文字区不贴边（边距 ≥ 画面宽 8%）

## 禁止

- 密密麻麻小字
- 多色正文混排
- 正文遮挡角色面部
