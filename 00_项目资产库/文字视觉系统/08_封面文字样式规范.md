# 封面文字样式规范

适用于小红书封面、抖音图文 P1、视频封面帧。

## 主标题

| 项 | 规范 |
|----|------|
| 字体 | 主标题字体（见 `01_字体规范.md`） |
| 颜色 | `#E98B45` 或 `#5A3B2E` |
| 描边 | 可选深棕细描边，增强可读 |
| 字数 | 建议 8～14 字，一行为主 |

## 副标题 / 钩子

- 字号为主标题 55% 左右
- 颜色 `#9A6A4A`
- 放在主标题下方或画面留白区

## 构图

- 文字占画面不超过 **40%**
- 人物与表情区域不被标题压住
- 竖版 3:4：标题优先上 1/3 或下 1/4 安全区

## 平台注意

| 平台 | 建议 |
|------|------|
| 小红书 | 痛点句 + 1 个情绪词，字大留白多 |
| 抖音图文 | 标题更短，可带 `#` 话题放发布文案而非封面 |
| 视频封面 | 与片头标题样式一致，3 秒内可读 |
