# 顺顺驾到 · AI 短视频创作流水线

## 每集核心生产链条

每一集按以下顺序推进，**不跳步、不并行乱序**：

```
灵感来源
    ↓
选题脚本
    ↓
分镜表
    ↓
分镜图提示词
    ↓
生成分镜图
    ↓
图生视频提示词
    ↓
生成视频
    ↓
剪映剪辑
    ↓
发布
    ↓
复盘
```
## 目录说明

| 目录 | 用途 |
|------|------|
| `00_项目资产库` | 角色/场景设定、固定提示词、音色、BGM、音效 |
| `01_灵感与选题库` | 爆款拆解、生活观察、AI 选题、选题池 |
| `02_选题脚本` | 单集选题与脚本（示例：`001_喂饭太烫`） |
| `03_分镜设计` | 分镜表、提示词、台词、剪辑节奏 |
| `04_分镜图生成` | 各镜头分镜图与提示词 |
| `05_图生视频` | 各镜头视频与提示词 |
| `06_剪映工程` | 素材、字幕、工程、成片 |
| `07_发布文案` | 多平台标题、封面、话题 |
| `08_数据复盘` | 播放数据与优化建议 |
| `09_模板库` | 各环节可复用模板 |
| `10_已完成作品归档` | 按阶段归档成片 |

## 新集命名

复制 `001_喂饭太烫` 对应文件夹，改为 `002_xxx`、`003_xxx`，在 `02`～`08` 各阶段同步维护，并按上方生产链条逐步填满。
