# 单篇图文页面结构

建议每篇控制在 **5～7 页**。

| 页码 | 名称 | 作用 |
|------|------|------|
| P1 | 封面页 | 一句话抓住痛点，让人想点开 |
| P2 | 小剧场引入 | 用顺顺一家演出真实场景 |
| P3 | 误区冲突页 | 展示家里常见错误认知 |
| P4 | 知识解释页 | 把核心知识点讲清楚 |
| P5 | 正确做法页 | 给宝妈可执行建议 |
| P6 | 总结互动页 | 一句话收尾，引导评论收藏 |

对应文件路径：`03_小剧场页面设计/{篇号}/`、`04_图文图片提示词/{篇号}/`、`05_图文成图/{篇号}/`
