# 标题样式规范

## 主标题

| 项 | 规范 |
|----|------|
| 字体 | 圆润粗黑体（见 `01_字体规范.md`） |
| 颜色 | 优先 `#E98B45` / `#5A3B2E` |
| 效果 | 可加轻微阴影；封面可加深棕描边 |
| 禁止 | 强烈发光、彩虹字、过度卡通 |

**适用：** 图文封面、视频开头、知识卡片标题

## 副标题

| 项 | 规范 |
|----|------|
| 字号 | 为主标题 50%～60% |
| 颜色 | `#9A6A4A` 或 `#5A3B2E` |
| 字重 | Medium，不加描边 |

**适用：** 封面副标、页面二级标题

## 层级

- 单页/单画面主标题不超过 1 行（12 字内更佳）
- 主副标题层级分明，不叠两种强调色
