# 信息卡片样式规范

适用于图文和视频里的知识点提示。

## 卡片底色

| 类型 | 色值 |
|------|------|
| 浅绿 | `#DCEAD7` |
| 浅蓝 | `#DDEBF7` |
| 浅黄 | `#FFF1C7` |
| 浅橙 | `#FFE0C2` |

## 文字

| 层级 | 颜色 |
|------|------|
| 卡片标题 | `#5A3B2E` |
| 正文 | `#444444` |
| 提醒 | `#D85C3A` |

## 样式

- 圆角矩形，圆角半径统一（建议 12～16px 比例感）
- 轻微阴影，不立体浮雕
- 信息不超过 **3 层**（标题 → 要点 → 备注）
- **每张卡片只讲一个重点**

## 禁止

- 单卡塞满整屏字
- 医院宣传栏式密集排版
